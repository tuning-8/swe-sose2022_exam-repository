void OpenMensa_Parser.XmlWriter.WriteXmlFile ( )
inline

Method that writes the whole .xml file.

A method, that generates an Instance of XmlTextWriter. It then calls the WriteOpenMensaStandardInformation method and the WriteMenuInformation method. WriteXmlFile() method is public , so it can be accessed for each potential instance of the XmlWriter class. After calling it, the whole .xml file will be (over-)written.
