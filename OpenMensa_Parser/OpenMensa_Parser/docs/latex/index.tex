\href{https://openmensa.org/\#5/48.893/11.194}{\texttt{ Open\+Mensa}} is a free database for canteens and cafeterias in Germany. The website provides the menus of over one thousand canteens. Open\+Mensa is completely free and collaborative developed.\hypertarget{index_autotoc_md2}{}\doxysection{Project overview}\label{index_autotoc_md2}
The aim of this Project is the creation of a website parser to primarily collect the menu data from the TU Freiberg canteen. The project is made in the context of the exam for the module \href{https://github.com/ComputerScienceLecturesTUBAF/SoftwareentwicklungSoSe2022_Projektaufgaben}{\texttt{ \textquotesingle{}Softwareentwicklung\textquotesingle{}}}. Detailed information about this project can be found in the \href{https://github.com/tuning-8/swe-sose2022_exam-repository/wiki}{\texttt{ Wiki}}\hypertarget{index_autotoc_md3}{}\doxysection{License}\label{index_autotoc_md3}
Currently the software is not under any specific license, but feel free to take inspiration from our code for creating a new parser to support the Open\+Mensa project.
